# 如何运行 DocChat 前端

## 前置条件

- Node.js 18+（推荐 20 LTS）
- npm 9+

## 第一次运行

```bash
# 进入项目目录
cd D:\docchat\new\docchat-frontend

# 安装依赖（仅需执行一次）
npm install

# 启动开发服务器
npm run dev
```

浏览器打开 **http://localhost:5173**

---

## 日常开发

```bash
npm run dev
```

Vite 会热重载，改完代码保存后浏览器自动刷新，不需要重启。

---

## 演示前检查

```bash
# TypeScript 类型检查 + 生产构建（验证无编译错误）
npm run build
```

构建产物在 `dist/`，构建成功说明代码没有 TS 错误。

---

## 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| 5173 端口被占用 | 上次没关干净 | 任务管理器结束 node 进程，或改 `vite.config.ts` 里的 `port` |
| 页面白屏 | 依赖没装 | 先跑 `npm install` |
| 样式全乱 | Tailwind 没生效 | 确认 `src/index.css` 第一行是 `@import 'tailwindcss'` |
